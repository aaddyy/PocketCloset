AdobeLabsMagicPerspectiveMode

Defines how the underlying image should have its perspective corrected. Change the mode property of the view instance using this enum type to modify the correction type. Note that some modes may look identical if the underlying image does not provide enough data to fully extract the scene’s perspective.
